Rocket League Random

Ever got bored of your current Rocket League car?
Simple solution: Generate a new loadout with Rocket League Random.
This webapp is a nice little tool to get a random Rocket League car without a lot of effort. Setup your owned DLCs and click one button. Only thing left to do: Pick the suggested parts in Rocket League!

Status: Not up to date.

Platform: Web

Undead Discord

This is a tool for managing Discord Bots.
When you are authenticated you can add, remove and setup different Discord bots to your liking and enjoy various different little tools and helpers to improve your Discord experience.
No need to download anything. Setup your bots through this webapp and enjoy them running 24/7 on a dedicated server.

Status: Work in Progress.

Platform: Web

APM Calculator

Actions per Minute (short: APM) is the number of key and mouse inputs a user performs per minute. If you have a passion for gaming and want to see just how fast you can go, an APM Calculator is a funny little tool that can help you show off to yourself.
Download this application, then follow the instructions and you'll be able to enjoy a lightweight, portable APM Calculator whenever you feel like it!

Status: No known issues.

Platform: Windows

Mouse Button Spammer

You want to click a lot but don't want to click a lot? Sounds like you need a Mouse Button Spammer!
This is a simple, lightweight tool for spamming your left mouse button as fast as you want. No installation required just set everything up to your liking and start the spam!

Status: No known issues.

Platform: Windows
